DOM Manipulation Exercises

Do not forget to watch the videos for each question

Question #1

Create a button that, when clicked, generates a random number between 0 and 100 and displays it inside a paragraph.

Question #2

Create a button that, when clicked, generates a random number between 0 and 100, displays it inside a paragraph using the DOM API, and shows the number in blue if it is odd, otherwise in red.

Question #3

Create a button that, when clicked, generates ten unique random numbers between 0 and 100 and displays them inside a paragraph using the DOM API. Add two checkboxes: Group (which groups the numbers into Odd and Even) and Sort (which sorts the numbers). Odd numbers should appear in blue and even numbers in red.

Question #4

Create five colored boxes (red, blue, green,magenta and orange). When each box is clicked, it should change the color of the ‘Hello World!!’ text to the corresponding color.

Question #5

Create a ‘+5px’ button. When clicked, it should increase the font size of the text ‘Sample Text Here’ by 5px and display the current font size on the page. The initial font size of p is default size.

Question #6

Create a page with a button labeled ‘Add a Number’. Each time the button is clicked, generate a random number between 0 and 100 and display it inside a newly created box. Each box should have a randomly generated background color. If the background color is dark (average RGB value < 150), the text color should be white; otherwise, it should be black. Append each new box inside the container.

Question #7

Create three boxes (100×60, you may use a table), each initially containing a random number between 0 and 10. The largest number(s) should be displayed with a larger, bold font and a #9F9 background. When the user clicks a box, its number should increase by 1.

Question #8

As in the demo, place three cat photos inside circular divs. When a cat photo is clicked, show a larger version of that photo below. Also remove the photo from the clicked circular area and change that circle’s border to dashed

Question #9

Create a simple click-based game. The page contains a 500×400 playground area and a Start button. When the Start button is clicked, the game should generate ten circular boxes at random positions inside the playground. Each circle must have a light-colored background and display a random number between 1 and 9. After generating the circles, the Start button should be hidden and a score display should appear, starting at zero. When the user clicks any circle, its number should be added to the score, and the circle should fade out with a CSS opacity(0), scale(2), translateY(-200px) transition and become unclickable (CSS "pointer-event:none"). Once all ten circles have been clicked, a ‘GAME OVER’ message should be shown.
